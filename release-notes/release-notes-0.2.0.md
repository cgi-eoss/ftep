# Release notes for F-TEP 0.2.0

F-TEP 0.2.0 is the initial alpha release.

## Processor services

The following processors have been introduced:

* LandCoverS2
* VegetationIndices
* Sentinel2Ndvi

## Application services

The following applications have been introduced:

* MonteverdiAppV2
* QGIS
* Sentinel2ToolboxV2
