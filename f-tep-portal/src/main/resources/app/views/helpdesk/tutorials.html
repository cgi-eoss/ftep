Helpdesk

The Forestry-TEP portal is divided up into various pages that provide particular functionality. The Explorer page allows users to search for satellite and other data, search and instigate services and to visualise their data using the various GUI applications. The Manage / Share page allows users to share various elements such as Projects, Databaskets, Jobs, Services and data. The My Account page allows users to view their account details and send requests such as for additional TEP-coins. If enabled, the Developer page allows Expert Users to develop, test and release their own processing services.

This initial release of the portal exposes a first set of both data and services. Within the coming releases the capabilities, in terms of data, services and resources will be increased.
